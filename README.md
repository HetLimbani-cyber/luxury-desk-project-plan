# Luxury Custom Reception Desk - MS Project Schedule

This repository contains the end-to-end manufacturing schedule for a luxury custom reception desk. It demonstrates practical project management workflows from initial handover to final client delivery.

## Key Features Demonstrated

* 
**Critical Path Management:** Actively tracking the critical path, which runs heavily through the finishing phase.


* 
**Lead Time Tracking:** Factoring in long-lead transit times for specialty materials like custom Italian leather hides.


* 
**Cure Periods:** Accounting for necessary lag times, such as the sealer curing process and final chemical topcoat cures.


* 
**Cross-Functional Coordination:** Scheduling dependencies across Engineering, CNC Machining, Assembly, and Finishing departments.



## Project Phases Outline

The work breakdown structure (WBS) includes 11 distinct phases:

* Phase 1-3: Initiation, Engineering & Drafting, Submittals & Approvals 


* Phase 4-6: Procurement & Inventory, CNC Machining & Milling, Sub-Assembly & Dry Fit 


* Phase 7-9: Finishing, Upholstery & Electrical, Final Integration & QA 


* Phase 10-11: Packaging & Logistics, Project Closure 



## Files Included

* **`Luxury Custom Reception Desk-v3.mpp`**: The native Microsoft Project file containing the full task list, dependencies, and Gantt chart.
* **`Luxury Custom Reception Desk-v3.pdf`**: A PDF export of the schedule for quick viewing.
